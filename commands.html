Beta Notice (Recommended for New Servers): Bombo Bot is currently in beta. For the cleanest and safest setup, we strongly recommend using it on a new/empty Discord server. If you run it on an existing server, choose Apply (no wipe) to avoid removing or overwriting any current channels, roles, or data. Only use any wipe option if you are intentionally starting fresh and have reviewed the preview first.
